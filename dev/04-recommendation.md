## Дополнительно для хостинга (https://ai-knowlege-css.thm.su)

После того как нейросеть даст код:

1. На сервере установи Node.js 20+, `pm2`
2. Загрузи папку проекта, выполни `npm install`
3. Собери React: `cd client && npm run build` (Vite соберёт в `dist/`)
4. В Koa добавь `koa-static` для папки `client/dist`
5. Запусти через `pm2 start server/index.js --name ai-rag`
6. Настрой nginx прокси на порт 3000 (или какой у Koa)
